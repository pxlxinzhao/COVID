# COVID-19 Tracker

COVID-19 Tracker is an Ionic hybrid app for live updates of COVID-19 statistics.

## Installation

Please install [node](https://nodejs.org/) and [ionic](https://ionicframework.com/) to get started.

## Usage

```javascript
npm install

ionic serve
```
